# Search @ thechelsuk

A search page that supports !bangs and @snaps, inspired by DuckDuckGo's !bangs but with a twist: @snaps allow you to search within a specific site using your default search engine with a site: filter.

Written in vanilla JavaScript, HTML, and CSS, this project is designed to be lightweight and a default homepage for browsers, with no dependencies and a focus on privacy.

The default search engine is DuckDuckGo (Lite), but you can set your preference in local browser storage using the provided dropdown list of engines.

The page also includes quick start examples to demonstrate the functionality of !bangs and @snaps.

You can view the live version at [search.thechels.uk/](https://search.thechels.uk/). Or pull the repo and open index.html in your browser of choice.
